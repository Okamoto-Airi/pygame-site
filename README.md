## 📌 遊び方のヒント
- 敵の攻撃に **3回当たるとゲームオーバー**
- ステージごとに操作方法が変わるので要チェック！
- キーボード操作のみ対応（PC推奨）

## ✨ 工夫した点
- **動画の上にプレイボタンを配置**し、直感的に操作できるように工夫
- **背景に画像を使用**することで、遊ぶ前からワクワク感を演出
- **重要なキーワードを色付きで強調**し、説明文を読みやすく整理
- **レスポンシブ対応**により、スマホやタブレットからも快適に閲覧可能
